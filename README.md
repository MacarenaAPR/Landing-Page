# 🧠 KineApps – Landing Page Kinesiología

Landing page desarrollada para un profesional de la salud (kinesiología), enfocada en captar pacientes, mostrar servicios y facilitar el contacto directo mediante WhatsApp.

---

## 🚀 Demo

👉 https://demo-landing-page-kineapp.vercel.app/ *(reemplaza con tu link real si quieres quedar pro)*

---

## 📌 Descripción

Proyecto web responsive orientado a:

* Mostrar servicios kinesiológicos
* Generar confianza con testimonios y presentación profesional
* Facilitar la conversión (contacto directo por WhatsApp)
* Adaptarse a dispositivos móviles y escritorio

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3 (Desktop + Mobile)
* Bootstrap 5
* Bootstrap Icons
* Google Fonts (Istok Web)

---

## 📱 Características

* Diseño responsive (mobile-first adaptado)
* Carrusel de servicios (Bootstrap)
* Scroll horizontal en secciones (UX moderna tipo app)
* Integración con WhatsApp
* Navegación simple y clara
* Secciones estructuradas:

  * Header (hero principal)
  * Problemas que resuelve
  * Servicios
  * Sobre el profesional
  * Proceso de atención
  * Testimonios
  * Contacto / Footer

## ⚙️ Instalación y uso

1. Clona el repositorio:

```bash
git clone https://github.com/MacarenaAPR/Landing-Page
```

2. Abre el proyecto en VS Code

3. Ejecuta con Live Server o abre `index.html` directamente en el navegador

---

## 🔒 Consideraciones de seguridad

* Se implementaron buenas prácticas básicas en HTML:

  * Uso de `rel="noopener noreferrer"` en enlaces externos
  * Meta tags de seguridad (CSP, referrer)
* No se manejan datos sensibles en frontend
* Recomendación: implementar backend seguro si se agregan formularios
